## Manage Parker Sales Team (CRUD Operation - EyePax Assignment) 

## Prerequisites

- Laravel 
- MySql
- JQuery
- Bootstrap

## Steps to config

1. Download source code from git 
2. Create a MySql database with the preferred name
3. Go to the source root folder and open the .ENV file set the database settings which are in line 11
4. Run "php artisan migrate" command, which will create the database table.
5. Run "php artisan serve" command.
6. Now you can open the http://127.0.0.1:8000 URL on your browser.
